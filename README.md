# Servo Arm Control System

## Product Description
This is the control software for our robotic arm product.

## Quick Start

### Arduino Setup
1. Open servo_control.ino in Arduino IDE
2. Upload to your Arduino board
3. Connect servos and joysticks according to pin definitions

### Python GUI Setup
1. Install Python 3.x
2. Run: pip install pyserial
3. Run: python servo_control.py

## Serial Commands
- 0 90  : Set Servo0=90
- 1 45  : Set Servo1=45
- 2 180 : Set Servo2=180
- 3 90  : Set Servo3=90
- 4 60  : Set Servo4=60
- 90    : Set ALL=90
- 90 90 90 90 90 : Batch set
